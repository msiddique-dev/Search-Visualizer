# AI Pathfinding Visualizer

This project is a Python-based visualizer for Artificial Intelligence search algorithms (Greedy Best-First Search and A* Search). It features a modern dark-mode UI and interactive grid elements.

## Prerequisites
To run this code, you need Python installed on your system along with the `pygame` library.

## Installation Instructions
1. Open your terminal or command prompt.
2. Install Pygame by running the following command:
   ```bash
   pip install pygame
